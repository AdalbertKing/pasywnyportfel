# Analysis definitions

This folder contains definitions of analyses: what should be calculated and how.

Each definition is a folder with:
- settings.csv — global parameters of the run: period, tax, CPI/FX paths, output options.
- portfolios.csv — portfolio list, synthetic/historical maps, rebalancing modes and INCLUDE flags.
- README.md — human description of that definition.

Default definitions:
- benchmark_1970_synth_usd_gross
- synth_vs_etf_2005
- user_template
