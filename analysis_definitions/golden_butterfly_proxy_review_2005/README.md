# golden_butterfly_proxy_review_2005

Task pomocniczy do porównania klasycznego proxy Golden Butterfly z wariantem fit proxy od 2005 r.

Uruchomienie:

```cmd
run_task.cmd golden_butterfly_proxy_review_2005
```

Task używa lokalnych map w:

```text
analysis_definitions\golden_butterfly_proxy_review_2005\maps\hist
analysis_definitions\golden_butterfly_proxy_review_2005\maps\synth
```

Nie jest uruchamiany automatycznie przez pełny start. Pełny start czyta tylko `analysis_definitions\startup_order.csv`.
