# user_template

Robocza piaskownica użytkownika. Można ją uruchomić przez:

```cmd
3-start_myanalise.cmd
```

Wynik trafi do `analysis_results\user_template__YYYYMMDD_HHMMSS`.

Do zapamiętania konkretnej analizy użyj:

```cmd
create_task.cmd moja_nazwa
run_task.cmd moja_nazwa
```

Nie traktuj `user_template` jako magazynu trwałych analiz.
